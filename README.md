# FIELDWORK — Cole's Landscape Co.

Landscaping business tracker with AI pricing advisor, voice commands, 
Google Sheets quotes, calendar sync, and net worth vault.

## Deploy to Vercel (Free — takes 3 minutes)

### Step 1: Push to GitHub
1. Go to github.com → New Repository → name it "fieldwork-app"
2. Upload all these files to the repo

### Step 2: Deploy on Vercel
1. Go to vercel.com → Sign up with Google/GitHub
2. Click "Add New Project"
3. Import your "fieldwork-app" GitHub repo
4. Framework: Vite
5. Click Deploy
6. Your app is live at fieldwork-app.vercel.app (or custom URL)

### Step 3: Add to iPhone Home Screen
1. Open your Vercel URL in Safari on your iPhone
2. Tap Share button (square with arrow up)
3. Tap "Add to Home Screen"
4. Name it "Fieldwork" → Add
5. It now works like a native app with your logo as the icon!

## Quick Local Dev
